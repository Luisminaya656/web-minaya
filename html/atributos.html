1.ATRIBUTOS GLOBALES

1.1 ID

Un ID en HTML es un identificador único que se utiliza para identificar un elemento en un documento HTML:

Se utiliza para vincular un elemento con un identificador de fragmento

Se utiliza en scripts

Se utiliza en hojas de estilo (con CSS)

Se utiliza para dar formato a la vista de operador con CSS

Se utiliza para manipular el modelo de objeto de documento con código JavaScript y DHTML

Se utiliza para modificar un elemento específico usando CSS o JavaScript

El valor del atributo id debe ser una cadena de caracteres opaca que no debe repetirse en todo el documento. Además, el valor no debe contener espacios en blanco.

A diferencia del ID, las clases se pueden asignar a varios elementos en una página o en todo el sitio web.

1.2 CLASS

Un class en HTML es un atributo que permite identificar un grupo de elementos HTML con características similares o iguales, como su color. El atributo class es útil para: Aplicar formato a la vista del operador con CSS Manipular el modelo de objeto de documento con código JavaScript y DHTML

Agrupar y destinar elementos en función de su propósito, función o apariencia

Para asignar una clase a un elemento HTML, se utiliza el atributo class. El valor del atributo class es una lista de elementos separados por espacios, que debe coincidir con el nombre de clase dado en el selector. Por ejemplo, <p class="intro">Esta es una introducción.</p>.

Los nombres de clases pueden contener caracteres ingleses, números, guiones y subrayados, pero el primer carácter no puede ser ni un número ni un guion. Se recomienda utilizar únicamente minúsculas.

Para aplicar un estilo a los elementos marcados con una clase, se puede usar CSS o JavaScript. Por ejemplo, .intro {font-weight: negrita;} es una regla CSS que pone en negrita el texto de los elementos con la clase "intro".

1.3 STYLE

En HTML, el atributo style especifica el valor de CSS que se aplica a un elemento HTML. Se utiliza para dar formato al elemento, como por ejemplo, para establecer su color, fuente o tamaño.

Para establecer el atributo style en un elemento, se puede utilizar el siguiente código:

Elemento span: style="font: Verdana; size: 48pt; color: #7f7f7f"

Tabla: style="font-family: Verdana; font-size 12pt; color: red"

La etiqueta style se integra en el encabezado del documento HTML, antes de los scripts. Se puede insertar cualquier número de elementos style en la sección HEAD.

El objeto style de DOM representa una sentencia de estilo individual. Se puede acceder a él a partir del document o desde el elemento para el cual se aplica el estilo.

El lenguaje de hojas de estilo (CSS) se encarga de la apariencia de las páginas web. Controla los colores, las fuentes y el diseño de los elementos de un sitio web.

2.ATRIBUTOS FORM

2.1 ACTIOM

En HTML, el atributo action del elemento <form> define la ubicación (URL) donde se envían los datos del formulario cuando se valida.

Por ejemplo, en el siguiente código HTML, el formulario envía los datos a la URL /my-handling-form-page utilizando el método post de HTTP:

Código: <form action="/my-handling-form-page" method="post"> </form>

Es una práctica estándar establecer siempre al menos los atributos action y method.

Si el campo action está vacío, la página actual se recarga con los parámetros con los datos del formulario.

2.2 METHOD

En HTML, el atributo method define el método HTTP que se utilizará para enviar los datos de un formulario. Los métodos más comunes son get y post:

get: Envía los datos como variables de URL.

post: Envía los datos como transacciones HTTP.

El atributo method se encuentra dentro de la etiqueta <form>. Los datos del formulario se envían a la página especificada en el atributo action.

2.3 TARGET

El atributo target en HTML es una herramienta que se utiliza para definir dónde se abre un enlace cuando un usuario hace clic en él. Se puede aplicar a las etiquetas A, AREA, BASE, FORM y LINK.

Los valores que puede tener el atributo target son:

_blank: Abre el enlace en una nueva ventana o pestaña

_self: Abre el enlace en la misma ventana

_parent: Abre el enlace en el marco principal

El atributo target puede ser útil para mantener a los usuarios en el sitio web, pero también puede ser contraproducente si se usa de forma incorrecta. Por ejemplo, puede dificultar la navegación de algunos grupos de usuarios. Por eso, se recomienda reducir su uso todo lo posible.

En HTML 5, el atributo target volvió a estar permitido, pero se recomienda utilizar el comando target_blank cuando salir de la página actual conlleve una pérdida de datos.

3.1 TYPE

Por ejemplo, se puede usar para crear un campo de texto, un campo de contraseña, un campo de correo electrónico, un campo de número, un campo de fecha, un campo de archivo, un botón, una casilla de verificación, un botón de radio, un área de texto, un campo de teléfono, un campo de color, un campo de rango, o un campo de imagen.

3.2 VALUE

El atributo value en HTML es un DOMString que contiene el valor actual del texto ingresado en un campo de texto. También se utiliza para establecer el valor de otros elementos, como botones, opciones, listas, medidores, barras de progreso y parámetros.

El atributo value tiene las siguientes funciones

Define el valor asociado con una entrada

Define el valor en el par nombre/valor que se envía al servidor al enviar un formulario

Especifica el texto que aparece en un botón

Define los datos que se envían al servidor o a un script cuando se hace clic en un botón

Define el valor predeterminado a ser mostrado al cargar la página

La sintaxis del atributo value para un botón es la siguiente: <botón> value="datos">etiqueta</botón>.

En esta sintaxis, datos es el valor que se envía al servidor o a un script cuando se hace clic en el botón, y etiqueta es el texto que se muestra en el botón.
