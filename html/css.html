¿QUE ES CSS?

CSS son las siglas de Cascading Style Sheets (hojas de estilo en cascada), un lenguaje de programación que se utiliza para diseñar y estructurar páginas web:

Permite controlar la presentación de los elementos de una página web, como su color, fuente, disposición, animaciones y efectos visuales

Separa el contenido del diseño, lo que facilita la gestión del aspecto visual de las páginas

Funciona junto con el lenguaje HTML, que se encarga del contenido básico de los sitios

El CSS tiene varios beneficios, entre ellos:

Mejorar la accesibilidad

Facilitar la creación de páginas web responsivas

Aumentar la eficiencia de las webs

Facilitar el mantenimiento de las páginas web

Aumentar la velocidad de carga de las páginas

Para aplicar CSS a una página web, se pueden utilizar hojas de estilos externas, internas o en línea:

Hojas de estilos externas: se vinculan al archivo CSS en el >head> de la página HTML con la etiqueta <link>

Estilos internos: se utilizan mediante la etiqueta <style> también dentro del <head>

Estilos en línea: se aplican directamente sobre elementos individuales a través del atributo style de CSS

El CSS fue desarrollado por W3C en 1996.

¿COMO UTILIZAR CSS CON HTML?

Para utilizar CSS en HTML, puedes hacerlo de varias formas:

Hojas de estilo internas

El código CSS se incluye dentro del documento HTML, ya sea al principio o en el código fuente. Para ello, debes:

Abrir la página HTML y localizar la etiqueta de apertura head

Después de la etiqueta head, escribir <style type="text/css">

Añadir las reglas CSS en una nueva línea

Escribir la etiqueta de cierre: </style>

Hojas de estilo externas

Se vincula el documento HTML con un archivo CSS externo. Este es el método más común y limpio para desarrollar páginas web. Para ello, debes:

Crear un nuevo archivo en el editor de texto con la extensión ".css"

Escribir las reglas CSS en el archivo

Vincular el archivo CSS al documento HTML con una etiqueta de "enlace" en la cabecera del documento HTML
CSS EN LINEA

CSS en línea es una forma de aplicar estilos directamente a los elementos de una página web HTML sin utilizar selectores. Para usar CSS en línea, se debe:

Agregar el atributo "style=" a la etiqueta de inicio

Escribir una propiedad-valor de CSS dentro de las comilla

Por ejemplo, para establecer un color de fondo salmón, se puede escribir "background: salmón"
