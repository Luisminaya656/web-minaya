1.ETIQUETAS DE DOCUMENTO.

1.1 HTML

HTML, acrónimo en inglés de HyperText Markup Language ('lenguaje de marcado de hipertexto'), hace referencia al lenguaje de marcado utilizado en la creación de páginas web. Este estándar que sirve de referencia del software que interactúa con la elaboración de páginas web en sus diferentes versiones. Define una estructura básica y un código (denominado código HTML) para la presentación de contenido de una página web, que incluye texto, imágenes, videos, juegos, entre otros elementos. Este estándar es gestionado por el World Wide Web Consortium (W3C) o Consorcio WWW, una organización dedicada a la estandarización de la mayoría de las tecnologías asociadas a la web, especialmente en lo relacionado con su escritura e interpretación. HTML se considera el lenguaje web más importante y su invención crucial para el surgimiento, desarrollo y expansión de la World Wide Web (WWW). Es el estándar que prevalece en la visualización de páginas web y es adoptado por todos los navegadores actuales.1​

El lenguaje HTML se fundamenta en la diferenciación como filosofía de desarrollo. Para añadir elementos externos a una página como imágenes, vídeos o scripts, no se incrustan directamente en el código de la página. En su lugar, se realiza una referencia a la ubicación de cada elemento mediante texto. De este modo, la página web contiene solamente texto, dejando al navegador web (intérprete del código) la labor de unir todos los elementos y visualizar la página final. Al ser un estándar, HTML pretende ser un lenguaje que permita que cualquier página web escrita en una determinada versión, pueda ser interpretada de manera uniforme (siguiendo el estándar) por cualquier navegador web actualizado.

1.2 HEAD

En HTML, head es la etiqueta que contiene la información de fondo de una página web, como el título, los enlaces al CSS y los metadatos. Esta información no se muestra en el navegador cuando se carga la página. Head es la primera de las dos partes en las que se estructura un documento HTML. La otra parte es body, que se encarga de desarrollar el cuerpo de la página web.

Algunos elementos que puede contener head son:



Título de la página (<title>) Enlaces al CSS Enlaces para personalizar favicon Metadatos, como quién lo escribió y palabras claves importantes También es posible añadir un código en head para aplicaciones avanzadas, como un "código de verificación" o un "código de seguimiento"
1.3 BODY

En HTML, el elemento <body> es la etiqueta que identifica la parte visible de un sitio web. Es el segundo elemento de un elemento <html> y solo puede haber un elemento <body> en un documento. Dentro del <body> se agregan las etiquetas para marcar los elementos visuales del sitio web, como el logotipo, los menús de navegación y el contenido principal.

Algunos atributos del elemento <body> son:

alink: Color del texto de los hipervínculos cuando se selecciona

background: URI de una imagen para usar como fondo

bgcolor: Color de fondo del documento

bottommargin: Margen de la parte inferior del <body>

leftmargin: Margen de la izquierda del <body>

link: Color del texto de los enlaces de hipertexto no visitados

Para acceder al elemento <body> se puede usar la propiedad document <body>.

2.ETIQUETAS DE META DATOS.

2.1 META

En HTML, una etiqueta meta es un elemento que proporciona información sobre un documento, como el contenido de una página web: Se coloca en el encabezado del código fuente, entre las etiquetas <html> y <body> Su forma es <meta name=»» content=»»/> Los atributos de la etiqueta meta son: name: Identifica un nombre de propiedad content: Especifica el valor de una propiedad scheme: Especifica un esquema para interpretar el valor de la propiedad http-equiv: Puede utilizarse en lugar del atributo name lang: Información sobre el idioma del contenido del elemento y del valor de sus atributos dir: Indica la dirección de texto y tablas Los metadatos son datos sobre datos que se utilizan para: Describir el contenido de una página web Ayudar a los motores de búsqueda a rastrear y clasificar las páginas web Especificar el conjunto de caracteres, la descripción de la página, las palabras clave, el autor del documento, y los ajustes de la ventana Los metadatos son una herramienta importante para el posicionamiento de una página web en los motores de búsqueda. Algunos ejemplos de etiquetas meta son: <meta charset="UTF-8"> <meta name="description" content="Free Web tutorials"> <meta name="keywords" content="HTML, CSS, JavaScript"> <meta name="author" content="John Doe"> <meta name="viewport" content="width=device-width, initial-scale=1.0">

2.2 STYLE

En HTML, style es un elemento que se encarga de definir la información de estilo de una página web. Se puede utilizar para dar formato a un elemento, como su color, fuente, tamaño, entre otros.

Para aplicar un estilo a un elemento HTML, se puede utilizar el atributo style. Por ejemplo, para establecer el estilo de un elemento span se puede utilizar el siguiente código:

Código style="font: Verdana; size: 48pt; color: #7f7f7f"

También se puede utilizar la etiqueta style en el encabezado del documento HTML, entre las etiquetas <HEAD> y </HEAD>. Esta etiqueta integra estilos globales de CSS y es compatible con varios atributos HTML. El CSS (hojas de estilo en cascada) es un lenguaje de marcado que se encarga de la apariencia de las páginas web. Además de controlar los colores, las fuentes y el diseño, también permite añadir efectos o animaciones.

2.3 TITLE

El título en HTML es un elemento de código que define el título de una página web y se muestra en la barra de título del navegador o en la pestaña de la página.

Título en HTML

2.4 BASE

En HTML, la etiqueta <base> establece la ruta base para los enlaces relativos de un documento. Esto significa que la URL indicada se añade a los enlaces relativos de la página.

<base>



Función:Establece la ruta base para los enlaces relativos de un documento HTML

Características:Solo puede haber un elemento <base> en un documento. Si se especifican varios elementos <base>, se utilizará solo la primera sección href y el primer valor target.

El lenguaje HTML es un lenguaje de marcado que sirve para crear páginas web estáticas. Su estructura se basa en etiquetas y atributos predefinidos que le indican al navegador cómo mostrar el contenido.
2.5 LINK

Un link en HTML, también conocido como hiperenlace, es una etiqueta que permite enlazar una página web con otra, o con un archivo o documento:

Se puede enlazar a una página del mismo directorio, de otro directorio, o de otro computador.

Se puede enlazar a una parte específica de una página.

Se puede enlazar a un archivo de un servidor FTP anónimo.

Para insertar un link en HTML se utiliza la etiqueta <a> (de anchor). El atributo "href" (abreviatura de "Hypertext REFerence") es un componente esencial del elemento <a> y determina el destino del enlace.

Para crear un link, se debe incluir el texto que se quiere convertir en enlace, y luego usar la etiqueta <a>. A continuación, se debe agregar el atributo "href" con la dirección web que se quiere que apunte el enlace. Algunos atributos que se pueden incluir en las etiquetas HTML link son:

rel: Describe la relación entre dos entidades relacionadas.

hreflang: Especifica el idioma del texto en el documento enlazado.

target: Da instrucciones de qué ventana debe abrir el enlace.

3.ETIQUETAS DE AGRUPACION

3.1 P

En HTML, la etiqueta <p> es la que se utiliza para crear párrafos en el texto. Las etiquetas que delimitan un párrafo son <p> y </p>, pero la segunda es opcional.

La etiqueta <p> crea una caja en bloque y puede contener cero o más elementos en línea. Los navegadores agregan automáticamente una línea en blanco antes y después de cada elemento <p>.

Para indicar la alineación del texto dentro de un párrafo, se pueden usar atributos específicos dentro de las etiquetas <p> y </p>. Por ejemplo, para alinear el texto a la izquierda, se usa el valor "left". Los otros valores son "right" para alinear a la derecha y "center" para centrar horizontalmente.

Para estilizar los párrafos, se puede usar CSS.

3.2 HR

La etiqueta <hr> en HTML es una línea horizontal que se utiliza para separar contenido en una página web. <hr> significa horizontal rule, que en español se traduce como "línea horizontal".

La etiqueta <hr> es una herramienta fundamental para crear divisores visuales que mejoren la organización y la legibilidad del texto. Se puede combinar con otros tags como encabezados h1- h6 y párrafos p.

Por lo general, la línea horizontal ocupa todo el ancho del bloque, por lo que no se puede poner nada "dentro" de ella.

Un ejemplo de uso de la etiqueta <hr> es:

<p>Este es un parrafo unicamente visual para ver el uso de la etiqueta hr </p> <hr/> <p>Este es el segundo parrafo, separado del primero por la etiqueta hr, que puede representarse mediante una línea horizontal.</p>
3.3 UL

En HTML, la etiqueta <ul> define una lista desordenada o "unordered list". Las etiquetas que definen una lista desordenada son <ul> y </ul>, ambas obligatorias.

Para crear una lista desordenada, se utiliza la etiqueta <ul> junto con la etiqueta <li>, que especifica cada uno de los elementos de la lista. Las etiquetas de <li> son <li> y </li>, siendo la de cierre opcional.

Para aplicar estilo a las listas, se puede utilizar CSS. Para crear listas ordenadas, se utiliza la etiqueta <ol>.

3.4 DIV

En HTML, un div es una etiqueta que define las divisiones lógicas de una página web. Su nombre proviene de la palabra inglesa division, que significa "sección", "área" o "división".

Los div se utilizan para:

Agrupar elementos HTML en un bloque

Centrar bloques de contenido

Crear efectos de columna

Crear diferentes áreas de color

Agrupar secciones de código para orientarlas con CSS

Cambiar el idioma de una sección particular de la página

Los div no tienen un significado semántico propio en HTML y no influyen en el diseño o el contenido de una página web. Por eso, se recomienda buscar soluciones semánticamente más adecuadas, como article, aside, header, main, nav o section.

La etiqueta div se utiliza de la siguiente manera:

<div> <p>Ejemplo de cómo funcionaria; la etiqueta div para divisiones.</p> <p>Puedes insertar el contenido que necesites.</p> <p>Esta seria la última línea de nuestro texto.</p> </div> Los atributos HTML más utilizados con div son class e id.
3.5 FIGURE

En HTML, una figura es una etiqueta que permite almacenar contenido independiente, como imágenes, diagramas, fotos, listas de códigos, entre otros. La etiqueta <figure> es una etiqueta de agrupación que debe contener el elemento de contenido y la etiqueta <figcaption>.

La etiqueta <figure> tiene las siguientes características:

Es una mejor práctica que usar solamente un contenedor div.

El contenido del elemento <figure> está relacionado con el flujo principal, pero su posición es independiente.

Si se elimina, no debería afectar el flujo del documento.

Tanto la etiqueta inicial como la etiqueta de cierre son obligatorias.

No tiene atributos que no sean los atributos globales, comunes a todos los elementos.
3.6 DIGCAPTION

En HTML, un figcaption es un elemento que se utiliza para representar un subtítulo o leyenda asociado a un contenido del elemento padre figure. El texto de un figcaption suele ser una aclaración o ampliación de la información que se encuentra en la figura o ilustración.

El elemento figcaption es opcional y puede colocarse como primer o último hijo.

Por otro lado, el elemento figure especifica contenido independiente, como ilustraciones, diagramas, fotos, listas de códigos, etc. El contenido del elemento figure está relacionado con el flujo principal, pero su posición es independiente del mismo.

4.ETIQUETAS TEXTUALES

4.1 A

En HTML, la etiqueta <a> es un elemento que permite crear un hipervínculo, o enlace, para navegar a otras páginas web, archivos, rutas, direcciones de correo, o cualquier otra URL. La "a" significa "ancla" porque crea un enlace que actúa como punto de anclaje para la navegación.

Para insertar un enlace en un texto HTML, se utiliza el comando <A> y se le especifican ciertos parámetros. Uno de estos parámetros es HREF, con el cual se le indica hacia dónde debe seguir el link.

Otro atributo importante de la etiqueta <a> es target, que puede tener el valor “self” o “_blank”. El valor “self” indica que el enlace se abrirá en la misma página, mientras que “_blank” indica que se abrirá en una pestaña nueva.

HTML, o Lenguaje de Marcado de Hipertexto, es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

4.2 SPAN

En HTML, un span es un contenedor en línea que se utiliza para agrupar elementos en una línea y aplicarles estilos. Las etiquetas de span son <span> y </span>.

Las etiquetas span son útiles para: Aplicar estilos al texto, Agregar atributos a una sección de texto, Manipular elementos con JavaScript.

A diferencia de los elementos de bloque, los span no crean una nueva línea o espacio. Por defecto, un elemento span no tiene ningún estilo aplicado, por lo que se suele usar en combinación con CSS.

La etiqueta span no transmite ningún significado semántico sobre el contenido que rodea. Por ejemplo, para resaltar palabras clave con fines de SEO, es más apropiado usar las etiquetas <strong> o <em>

4.3 SUB

En HTML, un sub es un fragmento de texto que se muestra en subíndice, es decir, más pequeño y más bajo que el texto principal. Para definir un subíndice en HTML se utiliza la etiqueta <sub>.

El texto en subíndice se puede utilizar para fórmulas químicas, como H2O.

Para definir un superíndice en HTML se utiliza la etiqueta <sup>.

4.4 SUP

En HTML, un sup es la etiqueta <sup>, que se utiliza para definir un fragmento de texto en superíndice. Esto significa que el texto se mostrará más alto y generalmente más pequeño que el resto del texto.

El elemento <sup> se debe usar solo por razones tipográficas, como en matemáticas o en abreviaturas francesas. Por ejemplo, la fórmula matemática f4 o las abreviaturas francesas Mlle, Mme o Cie.

Para producir un superíndice y un subíndice al lado del símbolo químico de un elemento, se necesita usar MathML.

4.5 CITE

En HTML, la etiqueta <cite> es un identificador que marca una referencia a una fuente o al autor de un texto citado. Las etiquetas que se utilizan son <cite> y </cite>, ambas obligatorias.

5.ETIQUETAS SEMANTICAS

5.1 NAV

En HTML, <nav> es una etiqueta que define una sección de navegación en un documento web. Esta sección suele contener enlaces o menús que permiten a los usuarios acceder a otras páginas o a diferentes partes del sitio web.

El elemento <nav> ayuda a organizar y estructurar el contenido de una página web de manera que sea más accesible para los visitantes.

Algunos ejemplos de secciones de navegación son: Menús, Tablas de contenido, Índices.

Para indicar las diferentes secciones o funcionalidades del sitio, se utilizan las etiquetas <nav> y </nav>. Las secciones se suelen indicar con una lista utilizando las etiquetas <ul> y <li>.

5.2 SECTION

En HTML, la etiqueta <section> es un elemento semántico que permite crear secciones independientes en una página web. Se utiliza para separar secciones de una página, como una sección de noticias o información de contacto.

Para usar secciones en HTML, se deben seguir los siguientes pasos:

Envolver los elementos relacionados que se quieren agrupar entre etiquetas <section>.

Añadir un encabezado a las secciones.

Es recomendable que las secciones tengan un título, con muy pocas excepciones.

La etiqueta <section> es una herramienta que ayuda a organizar y estructurar el contenido de manera semántica. Su uso adecuado mejora la accesibilidad y el SEO de la página, además de facilitar el mantenimiento y la comprensión del código.

Una regla general para determinar si el uso de la etiqueta <section> es correcto, es numerar los elementos. Si usando esta numeración el elemento <section> no pierde sentido, se puede entender que es un <section> realmente.
5.3 ARTICLE

En HTML, la etiqueta <article> define un bloque de contenido independiente y autónomo que puede ser distribuido y reutilizado de manera independiente.

La etiqueta <article> es una etiqueta semántica que se utiliza para especificar contenido que tiene un significado independiente del resto del sitio web. Por ejemplo, en una página de noticias o un blog, cada entrada sería un artículo dentro de una sección.

Algunas características de la etiqueta <article> son:

Siempre debe ir dentro de etiquetas <section>.

Se pueden poner tantos artículos como se quiera dentro de una <section>.
Afecta los atributos class, id y, en algunos casos, title.

Dentro de los artículos se pueden colocar cualquier etiqueta HTML.

No se debe anidar más de un artículo dentro de otro.
5.4 HEADER

El header en HTML es una etiqueta que define la sección de encabezado de una página web. Se utiliza para colocar los elementos más importantes de la navegación y de la marca, como el menú principal, el logo, los iconos de las redes sociales, entre otros.

El header es una parte fundamental de la web, ya que se repite en todas las páginas que la componen, lo que le da coherencia y facilita la navegación.

Para delimitar todo el encabezado, se debe abrir una etiqueta header dentro del body.

Algunos elementos que se pueden usar dentro de la etiqueta header son:

nav: Indica que hay una barra de navegación dentro

ul: Se usa para crear una lista desordenada

li: Son los ítems de una lista

a: Es una etiqueta para agregar enlaces a un texto
5.5 MAIN

En HTML, la etiqueta <main> es un elemento estructural que indica el contenido principal de una página web. Es como el eje central de la página, y se usa para indicar a los navegadores y desarrolladores que el contenido dentro de esta etiqueta es la información principal que buscan los usuarios.

La etiqueta <main> solo puede usarse una vez por cada página. Algunos lectores de pantalla reconocen esta etiqueta y ofrecen un atajo para que el usuario pueda saltar directamente al contenido de esta etiqueta.

5.6 FOOTER

El footer en HTML es una sección de la página web que se encuentra en la parte inferior del documento y que se traduce como pie de página. Su función es contener información que no es esencial para entender el contenido principal, pero que es importante para ofrecer una experiencia de usuario completa.

El footer en HTML se representa con la etiqueta <footer>. Por lo general, el pie de página es el mismo en todas las páginas de un sitio web y se incluye en la plantilla de la página.

El footer tiene las siguientes funciones:

Marcar el final de la página

Facilitar la navegación

Enlazar información para que los usuarios la encuentren fácilmente

Compilar una serie de datos que pueden haber pasado desapercibidos en la página

Para diseñar un footer, se recomienda que sea claro, conciso y fácil de leer. También es importante que los elementos del footer estén organizados de manera lógica y fácil de encontrar.
