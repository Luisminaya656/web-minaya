¿QUE ES JAVASCRIPT?

JavaScript es un lenguaje de programación que se utiliza para crear interactividad en las páginas web HTML:

Se puede incrustar directamente en la página web o referenciarse a través de un archivo .js.

El navegador ejecuta el script junto con el código HTML y CSS al visitar la página web.

JavaScript permite crear contenido dinámico, como menús desplegables, formularios, galerías de fotos, gráficos animados, entre otros.

JavaScript es compatible con otros lenguajes como PHP, Perl y Java.

Algunas de las ventajas de JavaScript son:

Es sencillo de aprender e implementar.

Se ejecuta directamente en el navegador web, sin necesidad de conectarse a un servidor.

Hay muchos recursos y foros disponibles para ayudar a los principiantes.

Sin embargo, JavaScript también tiene algunos límites, como:

Los distintos navegadores web interpretan el código JavaScript de forma diferente.

El código JavaScript que se ejecuta en el lado del cliente es vulnerable a la explotación por parte de usuarios irresponsables.
¿COMO UTILIZAR JAVASCRIPT CON HTML?

Ya hemos explorado cómo HTML, CSS y JavaScript son necesarios para construir páginas web funcionales y dinámicas. Una buena metáfora sería pensar en tu sitio web como un pastel de tres capas: HTML como base, CSS como capa intermedia y JavaScript como capa superior.

Primero se estructura el contenido de la página web con HTML, luego se añaden elementos de estilo y un diseño adecuado con CSS. A continuación, hay que añadir funciones dinámicas e interactivas con HTML.

Hay dos maneras de añadir JavaScript a HTML: la primera es incrustar el código JS en línea en HTML, mientras que la segunda requiere crear un archivo de JS externo.
