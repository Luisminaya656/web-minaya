¿QUE ES HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado que se utiliza para estructurar y mostrar el contenido de una página web. HTML es la base de blogs, redes sociales, ecommerces y otros sitios web. HTML define la estructura y el significado del contenido de una página web. Para ello, utiliza etiquetas y atributos para marcar el texto, las imágenes y otros elementos que componen la página. Algunas características de HTML son:

Define la jerarquía de títulos, el flujo de párrafos y la ubicación de imágenes.

Permite crear enlaces a otras páginas web, archivos, direcciones de correo, o cualquier otra URL.

Permite incrustar imágenes, videos, archivos de audio y otros elementos multimedia.

Permite crear tablas para organizar datos.

Permite crear formularios para recopilar información del usuario

HISTORIA HTML

La historia del HTML (Lenguaje de Marcas de Hipertexto) se remonta a 1989, cuando el físico Tim Berners-Lee, investigador del CERN, propuso un sistema de hipertexto para compartir documentos:

La primera versión de HTML se escribió en 1993 y contaba con 18 etiquetas para texto.

En 1991 se publicó el primer documento con la descripción de HTML, bajo el nombre de “HTML Tags”.

En 1993, el IETF (Internet Engineering Task Force) propuso oficialmente que HTML sea un estándar.

En 1999, HTML 4.01 se convirtió en el estándar oficial.

En 2006, el W3C se unió al proyecto WHATWG para unificar el desarrollo de HTML.

En 2007, el W3C mostró interés en el desarrollo de HTML5

¿QUE SON ETIQUETAS EN HTML?

Un elemento de HTML es un tipo de componente de documento de HTML (Hypertext Markup Lengua), uno de los diversos tipos de nodos de HTML ( hay también nodos de texto, nodos de comentario y otros). El documento de HTML está compuesto de un árbol de nodos HTML sencillo, como nodos de texto, y elementos de HTML, los cuales añaden semántica y formato a partes del documento (p. ej., texto en negrita, lo organiza en párrafos, listas y tablas o hiperenlaces e imágenes embebidas). Cada elemento puede tener atributos de HTML especificados. Los elementos también pueden tener contenido, incluyendo otros elementos y texto.

¿QUE SON ATRIBUTOS EN HTML?

Los atributos en HTML son palabras especiales que se añaden a las etiquetas de apertura para modificar el comportamiento o la visualización de los elementos HTML:


accept: Indica los tipos de archivos que acepta el servidor

align: Especifica el alineamiento horizontal del elemento

alt: Texto alternativo en caso de que la imagen no se pueda mostrar

autocomplete: Indica si los controles en un formulario pueden tener por defecto sus valores completados automáticamente

bgcolor: Color de fondo del elemento

color: Establece el color de texto

contenteditable: Indica si el contenido del elemento es editable

Existen diferentes tipos de atributos HTML, entre ellos:

Atributos requeridos: Son necesarios para que un tipo de elemento funcione correctamente

Atributos opcionales: Modifican la funcionalidad por defecto de un tipo de elemento

Atributos estándares: Son soportados por muchos tipos de elementos

Atributos de evento: Se utilizan para especificar guiones o script que se ejecutan bajo circunstancias concretas

REGLAS FUNDAMENTALES EN HTML

Algunas reglas fundamentales del HTML son:

Utilizar la sintaxis HTML5

Sangrar con dos espacios

No dejar espacios en blanco al final

Entrecomillar los atributos

Incluir los atributos en la etiqueta de apertura, no en la de cierre

No crear atributos propios, ya que esto puede confundir a los agentes de usuario

Los valores de los atributos deben ir entre comillas simples o dobles

Los nombres de los elementos no distinguen entre mayúsculas y minúsculas, pero se recomienda escribirlos en minúsculas

ETIQUETAS PARA HACER UN DOCUMENTO HTML

Para crear un documento HTML, se utilizan etiquetas que definen la estructura del documento y el contenido:

<!DOCTYPE html>: Define el tipo de documento

<html>: Inicia el documento

<head>: Contiene información general sobre el documento, como el título, metaetiquetas, scripts y estilos

<title> </title>: Inserta el título del documento

<body> </body>: Define el cuerpo del documento

<div>: Define un bloque de contenido o sección de la página

<a>: Crea enlaces a documentos externos o internos

<strong> </strhong>: Define texto importante en negrita

<br>: Inserta un salto de línea

<p> ⁢/p<: Representa un párrafo

<h1>…<h6>: Define títulos y subtítulos

<img> </img>: Inserta una imagen en el documento

<table> <caption> <thead> <tbody> <tfoot> <tr> <th> <td>: Define una tabla, su título, cabecera, cuerpo, pie, filas, celdas de encabezado y celdas de datos

< <ol> </ul> <li> </dl> <dt> <dd>: Crea listas ordenadas, desordenadas, de definiciones, define un concepto o posición en la lista de definiciones e indica la descripción de una definición

<object> </param>: Inserta documentos dentro de un documento HTML

Una etiqueta HTML está compuesta por tres partes: una etiqueta de apertura, el contenido y una etiqueta de cierre.
